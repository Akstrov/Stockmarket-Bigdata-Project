# Stock Market Prediction using Big Data & Sentiment Analysis

Real-time stock market prediction system using Reddit sentiment analysis and financial data.

## Team Members
- Member 1: Data Collection Engineer
- Member 2: Data Processing Engineer
- Member 3: ML Engineer
- Member 4: Integration Engineer

## Tech Stack
- Kafka, Spark, MongoDB
- Airflow, MLflow, DVC
- Docker & Docker Compose
- Python, TensorFlow

## Quick Start
```bash
docker-compose up -d
```

## Project Structure
- `data_collection/` - Reddit & Finance data collectors
- `data_processing/` - Spark streaming & data cleaning
- `ml_models/` - Sentiment analysis & prediction models
- `orchestration/` - Airflow DAGs & Dashboard
- `volumes/` - Persistent data storage
